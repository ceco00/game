# 🎮 Quiz Quest — Fun Learning Game for Kids

A free, interactive quiz game for kids covering dinosaurs, space, ocean life, and silly science facts. No installs, no accounts — just open and play!

🌐 **Live game:** [https://ceco00.github.io/game/](https://ceco00.github.io/game/)

---

## 🧠 Topics

| Topic | Questions |
|-------|-----------|
| 🦕 Dinosaurs | Necks, eggs, T-Rex facts & more |
| 🚀 Space | Moon, planets, astronauts & stars |
| 🐙 Ocean | Octopuses, sharks, coral reefs & whales |
| 😂 Silly Science | Sneezes, bones, bee vomit & more |

Each topic has **5 questions** with 4 answer choices and a fun fact after every answer.

---

## ✨ Features

- 🎯 Multiple choice questions with instant feedback
- 💡 Fun facts after every answer — right or wrong
- 📊 Score tracker and progress bar
- 🔀 Questions shuffled randomly every time
- 🏆 Result screen with emoji rating at the end
- 📱 Works on phones, tablets and computers
- 🌐 No internet required after first load (except Google Fonts)

---

## 🚀 How to Play

1. Open the game in any web browser
2. Pick a topic from the buttons at the top
3. Choose your answer for each question
4. Read the fun fact and keep going!
5. See your final score and play again

---

## 🛠️ Run It Locally

No setup needed — just download and open:

```bash
git clone https://github.com/ceco00/game.git
cd game
# Open index.html in your browser
open index.html
```

---

## 🌍 Host on GitHub Pages (Free)

1. Go to your repo on GitHub
2. Click **Settings → Pages**
3. Under **Source**, select `main` branch
4. Click **Save**
5. Your game is live at `https://ceco00.github.io/game/`
